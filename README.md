## Run Locally

Clone the project

```bash
  git clone https://github.com/saharfallahi/ProductManagementApp.git
```

Go to the project directory

```bash
  cd ProductManagementApp
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
